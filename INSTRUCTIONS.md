# Technologies
- ExpressJS
- Kafka + Zookeeper
- Socket.io
- MongoDB
- Docker

# Firstly, install things! (dev env only)
- run `yarn` inside root, `./backend` and `./services/messages`

# How to run it
- dev: `yarn dev:back` to start the Core API
- production: `docker-compose up -d` to start everything
